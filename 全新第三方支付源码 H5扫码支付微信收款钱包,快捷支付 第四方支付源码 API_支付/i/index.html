关于信达支付

北京信达支付信息技术有限公司（以下简称"信达支付"或公司）是专业的第三方支付机构，中国领先的行业支付解决方案提供商。公司于2016年在北京市新城区成立，注册资本100万元。信达支付是中国（北京）综试区首批战略合作伙伴，是目前业内新型的支付机构。

公司合作渠道均属于持有中国人民银行颁发的《支付业务许可证》，业务类型为互联网支付的持牌企业，基于跨境贸易及移动支付高速发展的现状，为满足各企业商家在交易环节中不断提高的收/付款需求，信达支付打造了以"跨境支付、移动支付、O2O支付、大数据风控"为业务核心的"全球化支付解决方案"。针对国内外商家垂直领域提供定制化支付解决方案，解决了互联网交易中"支付转化率、O2O交互、风险交易"等多项问题，极大缩短了跨境贸易商家的资金汇兑周期，提升了全球贸易企业的货币处理效率，助推了互联网交易产业的进一步完善。连连支付服务行业领域涉及电商、航旅、房产、消费分期等，累计服务商户15000余家，累计服务用户超过3亿人次。

温馨提示：北京信达支付信息技术有限公司仅提供支付服务，与商户和用户之间的交易无涉，不对用户和商户之间的交易及纠纷解决做任何的声明和承诺。用户需自行核实商户经营情况、产品和交易以及承担资金的安全、交易失败和投资损失等后果和责任。
